## Requirements

* **Editors:** vi, vim, emacs
* **Code:** W3C compliant, validated using [W3C-Validator](https://validator.w3.org/)
* **File Structure:**
    * CSS files in `styles` folder
    * Images in `images` folder
* **Restrictions:**
    * No `!important` or IDs (`#`) in CSS
    * No `img`, `embed`, `iframe` tags
    * No JavaScript
